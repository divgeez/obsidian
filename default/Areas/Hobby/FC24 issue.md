Now make sure your EA App is closed. Go to the EA website and download the EA Anticheat: [https://help.ea.com/en/help/pc/ea-anticheat/](https://help.ea.com/en/help/pc/ea-anticheat/) - Once you downloaded it, open it and "repair" your FC25 anticheat.

Once that is completed, do not start the EA App but instead go to "C:\Program Files\EA Games\EA SPORTS FC 25", find the file "EAAntiCheat.GameServiceLauncher" - double click to open. This should start to load EA/FC25 but it will stop and a window will popup, it will then ask you to log into your EA account. This is like a login for the anticheat. Once you login, the FC25 splash screen should show and start loading and hopefully it will launch the game. It did for me.
